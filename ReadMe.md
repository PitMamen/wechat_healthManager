## 项目介绍

医疗项目，使用腾讯云IM、TRTC来实现聊天和通话

使用小程序原生开发，搭配使用 vantUI 组件库

 代码地址在：
https://git.cdnlab.cn/projects/MCCLIENT/repos/health-manager-patient-wechat_v2/browse  

## 目录结构

```
packageIM //聊天和通话
pages //业务
```
